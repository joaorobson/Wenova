Function that renders keyboard or joystick help images.

This function renders keyboard or joystick help images if user select one of the options: 'keyboard' and 'joystick'; located at the menu 'Options'. Besides that, it will render buttons on the joystick test mode.

Check if user selected to enter on joystick test mode. If not, render the joystick or keyboard help image. Otherwise, render the test screen mode.
