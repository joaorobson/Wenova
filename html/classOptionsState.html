void OptionsState::update ( float  delta)
virtual

Update function.

This function identifies the pressed button and updates the values of the selected option.

Parameters
deltaa float variation to update menu state.

Check if the pressed button was SELECT (Keyboard) or B (Joystick). When user press one of these buttons and is on a submenu of the menu "Options" he return to the main menu of it's submenu.

Check if the pressed button was the cursor UP. When user press that button, the selected item goes to another one above it if the current item is not the first.

Check if the pressed button was the cursor DOWN. When user press that button, the selected item goes to another one below it if the current item is not the last.

Check if the pressed button was START (Keyboard) or A (Joystick). When user press one of these buttons and isn't on a submenu of the menu "Options" he enters to the submenu selected.

Check if user selected the option to change the screen resolution. Otherwise, the user selected the option to selected fullscreen enable.

Iterates all options of the menu "Options" and set the positions of texts on the menu "Options".

Implements State.
